CinemaScience Algorithms and Tools
==================================

CinemaScience includes algorithms and tools to facilitate analysis in situ and postprocessing, available on the `CinemaScience GitHub`_ page. The Cinema algorithm and tool projects include:

- `cinema_change_detection`_ A R command line tool that takes as input a Cinema database and produces a new Cinema database with change detection results using Myers et al. This tool also produces a change detection png image and json file to be used with the Cinema:Newsfeed.
- `cinema_lib`_ Python 3.6 library and command line tool for Cinema specifications A and D.
- `cinema_asqlpy`_ An apsw/SQLite virtual table interface to Cinema Spec A databases.
