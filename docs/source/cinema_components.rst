cinema_components Library
=========================

CinemaScience includes a library of viewer components that can be added by the user to create analysis and data specific viewers. These components include:

- PcoordSVG
- PcoordCanvas
- Glyph
- ImageSpread
- Query
- ScatterPlotSVG
- ScatterPlotCanvas
